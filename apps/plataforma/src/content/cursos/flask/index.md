Flask es un "micro" framework escrito en Python y concebido para facilitar el
desarrollo de aplicaciones Web bajo el patrón MVC.

Los siguientes contenidos forman parte de un curso impartido para
[OpenWebinars](https://openwebinars.net) en septiembre de 2017.

## Unidades

1. [Protocolo HTTP](curso/u01/)
2. [Programación web dinámica](curso/u02/)
3. [Patrón modelo-vista-controlador](curso/u03/)
